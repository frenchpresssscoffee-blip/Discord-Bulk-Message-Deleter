Discord Deleter
A web-based tool for bulk managing and deleting your Discord messages across servers and Dm.

Live site:https://discorddeleter.lovable.app/

What It Does
Discord Deleter lets you scan, filter, and mass-delete your own messages from any Discord server or direct message conversation. It runs entirely in the browser — nothing is stored on a server.

Features
Bulk message deletion — Delete your messages across servers and DMs in bulk with adjustable speed controls
Message scanning — Scan channels to find and preview your messages before deleting
Advanced filters — Filter messages by type (text, images, links, files, embeds), date range, keywords, and more
Server & DM support — Works with both guild (server) channels and direct message conversations
Live analytics — Real-time stats and charts showing deletion progress, message counts, and activity breakdowns
Message preview — Browse and inspect messages before taking action
Export options — Export scanned messages as PDF or JSON for your records
Search & sort — Search through scanned messages and sort by date or content
Dark/light theme — Toggle between dark and light mode
Multi-language support — Interface available in multiple languages
Sound effects — Optional audio feedback during scanning and deletion
Keyboard shortcuts — Quick actions via keyboard for power users
Token guide — Built-in instructions for retrieving your Discord token safely
DM auto-refresh — Automatically refreshes DM channels so new conversations appear without re-login
